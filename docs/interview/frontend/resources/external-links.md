## 💡 Tips for Using External Resources

```
DO:
✓ Focus on official documentation first
✓ Practice actively, not just read/watch
✓ Verify information from multiple sources
✓ Take notes in your own words

DON'T:
✗ Tutorial hell - watching without doing
✗ Rely solely on external explanations
✗ Skip fundamentals for advanced topics
✗ Memorize without understanding
```

## 🎯 How to Use

### Flashcards (Spaced Repetition)

```
RECOMMENDED APPS:
• Anki (Free, powerful)
• Quizlet (Easy to use)
• RemNote (Note + SRS)

IMPORT FORMAT:
Front: [Question]
Back: [Answer]

REVIEW SCHEDULE:
• New cards: 20/day max
• Reviews: All due cards
• Best time: Morning + before sleep
```

### Cheatsheets (Quick Reference)

```
USE CASES:
• Quick lookup during practice
• Before interviews (last-minute review)
• When stuck on syntax

DO NOT:
• Memorize without understanding
• Use during coding tests
• Replace actual learning
```

## ❓ Frequently Asked Questions

### "Tại sao frontend cần biết CS foundations?"

Big Tech interviews không chỉ test coding, mà còn đánh giá:
- Problem-solving approach
- Optimization thinking
- Technical communication
- Foundation understanding

### "Cần học sâu đến mức nào?"

Mức độ vừa đủ cho phỏng vấn:
- Hiểu concepts, không cần implement từ scratch
- Biết trade-offs giữa các data structures
- Phân tích được complexity
- Nhận ra patterns trong problems

### "Bắt đầu từ đâu nếu chưa có background CS?"

1. Start với computer-science-basics.md
2. Focus vào concepts liên quan JavaScript
3. Practice với frontend-related problems
4. Don't try to memorize, understand the "why"

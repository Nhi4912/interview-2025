## Google

### Interview Process
```
1. Recruiter Screen (30 min)
2. Technical Phone Screen (45-60 min)
   - 1-2 coding problems
3. Onsite (4-5 rounds, virtual or in-person)
   - 2 Coding rounds
   - 1 Frontend round
   - 1 System Design (for senior)
   - 1 Googleyness & Leadership
4. Hiring Committee Review
5. Team Matching
6. Offer
```

### Frontend Focus
- **DOM manipulation** in vanilla JS
- **Performance optimization**
- **Accessibility**
- **LeetCode Medium-Hard**

### Tips
- Practice on whiteboard/doc
- Explain thought process clearly
- Google uses structured interviews - prepare for specific formats

## Meta

### Interview Process
```
1. Recruiter Screen
2. Technical Phone Screen (45 min)
   - 2 coding problems
3. Onsite (4-5 rounds)
   - 2 Coding rounds
   - 1 Frontend-specific round
   - 1 System Design
   - 1 Behavioral
4. Team Matching (now required before offer)
5. Offer
```

### Frontend Focus
- **React deep dive** (they created it!)
- **System Design** - News Feed, Chat
- **Product sense**
- **Move fast** mentality

### Tips
- Practice React coding challenges
- Know React internals well
- Be ready for product-focused system design

## Amazon

### Interview Process
```
1. Online Assessment
   - 2 LeetCode problems
   - Work simulation
   - Work style survey
2. Phone Screen (60 min)
   - 1 coding + behavioral
3. Onsite Loop (4-5 rounds)
   - Each round: Coding + LP questions
   - 1 Bar Raiser round
4. Offer (or not - strict bar)
```

### Leadership Principles Focus
**EVERY question ties back to LPs**

| Priority LPs | Mô Tả |
|--------------|-------|
| Customer Obsession | Always |
| Ownership | Always |
| Dive Deep | Technical |
| Deliver Results | Always |
| Earn Trust | Always |

### Tips
- Prepare **2+ stories per LP**
- Use STAR method strictly
- Be specific, data-driven
- Bar Raiser can veto - they look for culture fit

## Microsoft

### Interview Process
```
1. Recruiter Screen
2. Phone Screen (45-60 min)
   - Coding or system design
3. Onsite (4-5 rounds)
   - 2 Coding rounds
   - 1 System Design
   - 1 Behavioral
   - 1 Hiring Manager
4. Offer
```

### Frontend Focus
- **React component design**
- **Accessibility** (strong focus)
- **System design** với collaboration aspects
- **Growth mindset** trong behavioral

### Tips
- Microsoft interviews are more collaborative
- They want to see how you think
- Accessibility knowledge is valued
- Show growth mindset and learning ability

## Apple

### Interview Process
```
1. Recruiter Screen
2. Technical Phone Screen
3. Onsite (6-8 rounds - longest!)
   - Multiple coding rounds
   - System design
   - Design critique
   - Behavioral/Team fit
4. Offer
```

### Frontend Focus
- **Attention to detail**
- **Performance và polish**
- **Design sensibility**
- **Privacy và security**

### Tips
- Apple interviews are LONG - pace yourself
- They value passion for Apple products
- Show attention to detail in code
- Privacy is important to Apple

---

## Netflix

### Interview Process
```
1. Recruiter Screen
2. Technical Phone Screen
3. Onsite (4-5 rounds)
   - Coding (less emphasis)
   - System Design (major focus)
   - Culture fit (multiple rounds)
4. Reference checks (thorough)
5. Offer
```

### Culture Focus
Netflix Culture Deck values:
- **Judgment**
- **Communication**
- **Curiosity**
- **Courage**
- **Passion**
- **Selflessness**
- **Innovation**
- **Inclusion**
- **Integrity**
- **Impact**

### Tips
- Netflix hires almost exclusively senior+ roles
- Culture fit is VERY important
- They expect you to be autonomous
- "Freedom and Responsibility" is key

## Common Mistakes

### All Companies
- Not researching company culture
- Generic answers
- Not asking questions
- Poor communication

### Google
- Not explaining thought process
- Rushing to code

### Meta
- Not knowing React deeply
- Missing product thinking

### Amazon
- Ignoring Leadership Principles
- Vague answers without STAR

### Microsoft
- Not showing growth mindset
- Ignoring accessibility

### Apple
- Lack of attention to detail
- Not showing passion for products

### Netflix
- Not being senior enough
- Poor culture fit

## 🧠 Intuition / Tư Duy

**Analogy:** Tìm 3 người trong nhóm có tổng tuổi = 0 (2 người dương, 1 người âm để cân bằng). Nếu biết người đầu tiên (fix nums[i]), bài toán thành **Two Sum = -nums[i]** trên phần còn lại.

**Pattern Recognition:**

- Signal: "find triplets that sum to 0", "three numbers" → **Sort + Fix one + Two Pointers**
- Sort trước → Two Pointers O(n) thay vì HashMap, dễ skip duplicates
- Ba điểm skip duplicates: tại i, tại lo, tại hi

**Visual — nums = [-4,-1,-1,0,1,2] (sorted):**

```
i=1, nums[i]=-1, target=1
  lo=2(-1), hi=5(2): sum=-1+2=1 → FOUND [-1,-1,2] ✓
    skip dupes: lo→3, hi→4
  lo=3(0), hi=4(1):  sum=0+1=1  → FOUND [-1,0,1] ✓

i=2, nums[i]=-1: same as i=1 → skip (duplicate)
Result: [[-1,-1,2], [-1,0,1]]
```

## Problem Description

Given integer array `nums`, return all unique triplets `[nums[i], nums[j], nums[k]]` where `i != j != k` and `nums[i] + nums[j] + nums[k] == 0`. No duplicate triplets allowed.

```
Example 1: nums = [-1,0,1,2,-1,-4] → [[-1,-1,2],[-1,0,1]]
Example 2: nums = [0,1,1]          → []
Example 3: nums = [0,0,0]          → [[0,0,0]]
```

Constraints:

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

## 🗣️ Interview Script / Kịch Bản Phỏng Vấn

### Step 1 — Understand / Hiểu Đề (1-2 min)

> "Let me make sure I understand. We have an integer array.
> We need all unique triplets that sum to zero — no duplicate triplets.
> Clarification: Can numbers be negative? Can we modify the input array?"

### Step 2 — Match & Plan / Nhận Dạng & Lên Kế Hoạch (2-3 min)

> "Brute force is O(n³) — three nested loops with a Set for dedup.
> But if I sort first, I can fix one element and use Two Pointers for the other two — O(n²).
> Sorting also makes duplicate-skipping easy with simple comparisons. Should I proceed?"

### Step 3 — Implement / Viết Code (5-7 min)

> "Sort the array first.
> Outer loop: fix nums[i], skip if same as nums[i-1].
> Inner: two pointers lo=i+1, hi=n-1. Adjust based on sum vs 0.
> When found, skip duplicate lo/hi values."

### Step 4 — Review / Kiểm Tra (1-2 min)

> "Trace: [-1,0,1,2,-1,-4] → sorted: [-4,-1,-1,0,1,2].
> i=0(-4): lo=1,hi=5 → sum=-4-1+2=-3<0 → lo++... no match.
> i=1(-1): lo=2,hi=5 → sum=-1-1+2=0 → found [-1,-1,2]. Skip dupes.
> lo=3,hi=4 → sum=-1+0+1=0 → found [-1,0,1]. Correct."

### Step 5 — Evaluate / Đánh Giá (1 min)

> "Time: O(n²) — sort O(n log n) + nested two-pointer O(n²). Space: O(1) extra (ignoring output).
> Edge cases: all zeros [0,0,0], no valid triplets, all positive numbers.
> Early termination: if nums[i] > 0, break — no more negative numbers to balance."

## 📝 Interview Tips / Mẹo Phỏng Vấn

1. **Sort first** — enables Two Pointers AND makes duplicate-skipping trivial / Sort là bắt buộc
2. **Three skip points** — skip outer i dupes, skip lo dupes, skip hi dupes / 3 điểm skip
3. **Early termination** — if `nums[i] > 0` break, no sum=0 possible / Dừng sớm khi i > 0
4. **Explain reduction** — "Fix one, reduce to 2Sum" / Nói rõ cách giảm từ 3Sum xuống 2Sum
5. **Time complexity** — O(n²) not O(n³); sort dominated by two-pointer / O(n²) tổng
6. **Variant** — 3Sum Closest: track min distance instead of exact match / Biến thể tìm gần nhất

## Solutions

```typescript
/**
 * Solution 1: Brute Force — Three Nested Loops
 * Time: O(n³) — check every triplet combination
 * Space: O(n) — Set for dedup
 */
function threeSumBrute(nums: number[]): number[][] {
  const seen = new Set<string>();
  const result: number[][] = [];
  nums.sort((a, b) => a - b);

  for (let i = 0; i < nums.length - 2; i++) {
    for (let j = i + 1; j < nums.length - 1; j++) {
      for (let k = j + 1; k < nums.length; k++) {
        if (nums[i] + nums[j] + nums[k] === 0) {
          const key = `${nums[i]},${nums[j]},${nums[k]}`;
          if (!seen.has(key)) {
            seen.add(key);
            result.push([nums[i], nums[j], nums[k]]);
          }
        }
      }
    }
  }
  return result;
}

/**
 * Solution 2: Sort + Two Pointers (Optimal)
 * Time: O(n²) — outer loop O(n) × inner two-pointer O(n)
 * Space: O(1) — only pointers (ignoring sort and output)
 */
function threeSum(nums: number[]): number[][] {
  const result: number[][] = [];
  nums.sort((a, b) => a - b);
  const n = nums.length;

  for (let i = 0; i < n - 2; i++) {
    if (nums[i] > 0) break;
    if (i > 0 && nums[i] === nums[i - 1]) continue;

    let lo = i + 1,
      hi = n - 1;
    while (lo < hi) {
      const sum = nums[i] + nums[lo] + nums[hi];
      if (sum === 0) {
        result.push([nums[i], nums[lo], nums[hi]]);
        while (lo < hi && nums[lo] === nums[lo + 1]) lo++;
        while (lo < hi && nums[hi] === nums[hi - 1]) hi--;
        lo++;
        hi--;
      } else if (sum < 0) {
        lo++;
      } else {
        hi--;
      }
    }
  }
  return result;
}

// === Test Cases ===
console.log(threeSum([-1, 0, 1, 2, -1, -4])); // [[-1,-1,2],[-1,0,1]]
console.log(threeSum([0, 1, 1])); // []
console.log(threeSum([0, 0, 0])); // [[0,0,0]]
```

# 3Sum / Ba Tổng Bằng 0

> **Track**: Shared | **Difficulty**: 🟡 Medium | **Pattern**: Sort + Two Pointers
> **Frequency**: 🔥 Tier 1 — Extension of Two Sum, extremely common
> **See also**: [Two Sum](./04-two-sum.md) | [3Sum Closest](https://leetcode.com/problems/3sum-closest/) | [4Sum](https://leetcode.com/problems/4sum/)

## 🧠 Intuition / Tư Duy

**Analogy:** Tìm 3 người trong nhóm có tổng tuổi = 0 (tức là: 2 người có tuổi dương, 1 người tuổi âm để cân bằng). Nếu biết người đầu tiên (fix nums[i]), bài toán trở thành **Two Sum = -nums[i]** trên phần còn lại.

**Pattern:** Sort → fix one element → Two Pointers on the rest

- Sort trước → Two Pointers O(n) thay vì HashMap O(n)
- Skip duplicates để tránh duplicate triplets

**Visual:**

```
nums = [-4,-1,-1,0,1,2]  (sorted)
        ↑ fix i=0

i=1, nums[i]=-1, target=1
  lo=2(-1), hi=5(2): sum=-1+2=1 == 1 → FOUND [-1,-1,2] ✓
    skip dupes: lo++ until ≠-1, hi-- until ≠2
  lo=3(0),  hi=4(1): sum=0+1=1  == 1 → FOUND [-1,0,1] ✓

i=2, nums[i]=-1: same as i=1 → skip (duplicate)
Result: [[-1,-1,2], [-1,0,1]]
```

## Problem Description

**LeetCode #15.** Given integer array `nums`, return all unique triplets `[nums[i], nums[j], nums[k]]` such that `i != j != k` and `nums[i] + nums[j] + nums[k] == 0`. No duplicate triplets.

```
Example 1: nums = [-1,0,1,2,-1,-4] → [[-1,-1,2],[-1,0,1]]
Example 2: nums = [0,1,1]          → []
Example 3: nums = [0,0,0]          → [[0,0,0]]
```

## 📝 Interview Tips / Mẹo Phỏng Vấn

1. **Sort là bước bắt buộc** — enables Two Pointers AND makes duplicate-skipping easy với `nums[i] === nums[i-1]`
2. **Ba điểm skip duplicates** — tại vòng ngoài `i`, và cả `lo`/`hi` sau khi tìm được triplet
3. **Early termination** — nếu `nums[i] > 0` thì không thể tìm được sum = 0 → break ngay
4. **Explain the reduction** — "Fix nums[i], reduce to 2Sum on [i+1..n-1] with target = -nums[i]"
5. **Time complexity** — O(n²) not O(n³); sort is O(n log n) dominated by O(n²) two-pointer
6. **Biến thể** — 3Sum Closest: track `Math.abs(sum - target)` thay vì `sum === 0`

## Solutions

```typescript
// Solution 1: Sort + Two Pointers — O(n²) time, O(1) space ← OPTIMAL
function threeSum(nums: number[]): number[][] {
const result: number[][] = [];
nums.sort((a, b) => a - b);
const n = nums.length;

for (let i = 0; i < n - 2; i++) {
if (nums[i] > 0) break; // all positive → no triplet
if (i > 0 && nums[i] === nums[i - 1]) continue; // skip outer duplicates

    let lo = i + 1, hi = n - 1;

    while (lo < hi) {
      const sum = nums[i] + nums[lo] + nums[hi];

      if (sum === 0) {
        result.push([nums[i], nums[lo], nums[hi]]);
        while (lo < hi && nums[lo] === nums[lo + 1]) lo++; // skip lo dupes
        while (lo < hi && nums[hi] === nums[hi - 1]) hi--; // skip hi dupes
        lo++;
        hi--;
      } else if (sum < 0) {
        lo++;
      } else {
        hi--;
      }
    }

}

return result;
}

// Solution 2: Brute Force — O(n³) time, O(n) space ← baseline only
function threeSumBrute(nums: number[]): number[][] {
const seen = new Set<string>();
const result: number[][] = [];
nums.sort((a, b) => a - b);

for (let i = 0; i < nums.length - 2; i++) {
for (let j = i + 1; j < nums.length - 1; j++) {
for (let k = j + 1; k < nums.length; k++) {
if (nums[i] + nums[j] + nums[k] === 0) {
const key = `${nums[i]},${nums[j]},${nums[k]}`;
if (!seen.has(key)) { seen.add(key); result.push([nums[i], nums[j], nums[k]]); }
}
}
}
}

return result;
}
```

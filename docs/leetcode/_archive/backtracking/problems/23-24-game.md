## 🧠 Intuition / Tư Duy

**Analogy:** Giống bài toán "thử tất cả cách kết hợp" — chọn 2 số bất kỳ, áp dụng phép tính, thay thế chúng bằng kết quả, lặp lại. Bài toán thu nhỏ dần từ 4 số → 3 → 2 → 1.

**Pattern Recognition:**

- Signal: "try all combinations of operations on numbers" → **Backtracking / DFS**
- Key: chọn 2 số từ tập, tính kết quả (4 ops, cả hoán vị), tiếp tục với n-1 số
- Floating point: dùng `Math.abs(result - 24) < 1e-6` tránh lỗi precision

**Visual — Reduce 4 numbers to 1:**

```
[3, 3, 8, 8]
Pick 8, 3 → 8/3 = 2.666... → [3, 8, 2.666]
Pick 8, 3 → 8-3 = 5       → [5, 2.666]
Pick 5, 2.666 → 5*... no
Pick 3, 8/3 → 3*(8/3)=8   → [3,8] → 3-8 no...
Solution: 8/(3-8/3) = 8/(1/3) = 24 ✓

Try ALL pairs × ALL ops (including a-b AND b-a, a/b AND b/a)
```

## Problem Description

Given 4 integers (each 1–9), determine if you can get exactly `24` using `+`, `-`, `*`, `/` and parentheses in any order. Each number must be used exactly once. ([LeetCode 679](https://leetcode.com/problems/24-game))

**Example 1:** `[4, 1, 8, 7]` → `true` (e.g. `(8-4)*(7-1)=24`)
**Example 2:** `[1, 2, 1, 2]` → `false`

Constraints: `cards.length == 4`, `1 <= cards[i] <= 9`

## 📝 Interview Tips

1. **Clarify**: "Có phép chia không? Division by zero?" / Yes division, handle div-by-zero (skip if divisor ≈ 0)
2. **Key insight**: "Chọn 2 số bất kỳ → tính → đệ quy" / Pick any 2, apply op, recurse with n-1 numbers
3. **Symmetric ops**: "a-b ≠ b-a, a/b ≠ b/a → thử cả hai" / Subtraction and division are not commutative
4. **Float precision**: "Dùng epsilon 1e-6 thay vì === 24" / Always compare floats with epsilon
5. **Base case**: "1 phần tử, kiểm tra ≈ 24" / Single number: check if it equals 24

## Solutions

```typescript
/**
 * Solution 1: Backtracking — pick 2 numbers, apply all ops, recurse
 * Time: O(1) — bounded: 4 cards, constant branching factor
 * Space: O(1) — at most 4 levels deep
 */
function judgePoint24(cards: number[]): boolean {
  const EPS = 1e-6;

  function dfs(nums: number[]): boolean {
    if (nums.length === 1) return Math.abs(nums[0] - 24) < EPS;

    for (let i = 0; i < nums.length; i++) {
      for (let j = 0; j < nums.length; j++) {
        if (i === j) continue;
        const a = nums[i],
          b = nums[j];
        // Remaining numbers after removing i and j
        const rest = nums.filter((_, idx) => idx !== i && idx !== j);

        const candidates: number[] = [a + b, a - b, a * b];
        if (Math.abs(b) > EPS) candidates.push(a / b);

        for (const val of candidates) {
          if (dfs([...rest, val])) return true;
        }
      }
    }
    return false;
  }

  return dfs(cards.map(Number));
}

/**
 * Solution 2: Cleaner version — iterate over all pairs once (avoid duplicates)
 * Time: O(1) — same constant bound
 * Space: O(1) — 4 levels recursion max
 */
function judgePoint24V2(cards: number[]): boolean {
  const EPS = 1e-6;

  function solve(nums: number[]): boolean {
    if (nums.length === 1) return Math.abs(nums[0] - 24) < EPS;

    for (let i = 0; i < nums.length; i++) {
      for (let j = i + 1; j < nums.length; j++) {
        const a = nums[i],
          b = nums[j];
        const rest = nums.filter((_, k) => k !== i && k !== j);

        // Both orders for non-commutative ops
        const results: number[] = [
          a + b,
          a * b, // commutative
          a - b,
          b - a, // both orders
        ];
        if (Math.abs(b) > EPS) results.push(a / b);
        if (Math.abs(a) > EPS) results.push(b / a);

        for (const val of results) {
          if (solve([...rest, val])) return true;
        }
      }
    }
    return false;
  }

  return solve(cards.map(Number));
}

// === Test Cases ===
console.log(judgePoint24([4, 1, 8, 7])); // true  → (8-4)*(7-1)=24
console.log(judgePoint24([1, 2, 1, 2])); // false
console.log(judgePoint24([3, 3, 8, 8])); // true  → 8/(3-8/3)=24
console.log(judgePoint24([1, 1, 1, 1])); // false
console.log(judgePoint24V2([4, 1, 8, 7])); // true
```

## 🧠 Intuition / Tư Duy

**Analogy:** Giống bài 3Sum nhưng thêm 1 lớp — cố định 2 phần tử ngoài bằng 2 vòng lặp, rồi dùng 2 con trỏ cho phần còn lại. Sort trước để tránh duplicate và cho phép di chuyển con trỏ có hướng.

**Pattern Recognition:**

- Signal: "tìm quadruplets có tổng = target" + "không dùng chỉ số trùng" → **Sort + Fix 2 + Two Pointers**
- Mỗi lớp lồng thêm 1: 4Sum = O(n³), 3Sum = O(n²), 2Sum sorted = O(n)
- Dedup: skip phần tử trùng liên tiếp sau khi sort ở mỗi cấp độ

**Visual — nums=[-2,-1,0,0,1,2], target=0:**

```
Sorted: [-2, -1, 0, 0, 1, 2]

Fix i=-2, j=-1: lo=0, hi=2 → sum=-2-1+0+2=-1 < 0 → lo++
Fix i=-2, j=-1: lo=1, hi=2 → sum=-2-1+0+2=-1 < 0 → lo++  (wait: lo=0,hi=2 → val=0+2=2)
Fix i=-2, j=0:  lo=0(idx2), hi=5(idx5=2) → sum=-2+0+0+2=0 ✅ → record
Fix i=-1, j=0:  lo=2, hi=5  → sum=-1+0+0+2=1 > 0 → hi--
...

Dedup rule: if nums[i]==nums[i-1] && i>startIdx → skip
```

## Problem Description

Given an integer array `nums` and a `target`, return all unique quadruplets `[a,b,c,d]` such that `a+b+c+d == target`. The same index cannot be used twice; the result must have no duplicate quadruplets.

```
Example 1: nums=[1,0,-1,0,-2,2], target=0  → [[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]
Example 2: nums=[2,2,2,2,2], target=8       → [[2,2,2,2]]
```

Constraints: `1 <= nums.length <= 200`, `-10^9 <= nums[i] <= 10^9`, `-10^9 <= target <= 10^9`

---

## 📝 Interview Tips

1. **Clarify**: "Kết quả có thể có bộ 4 trùng không?" / Confirm no duplicate quadruplets in output.
2. **Brute force**: 4 vòng lặp lồng nhau O(n⁴) + Set dedup — explain thinking, then optimize.
3. **Optimize**: Sort O(n log n), fix 2 phần tử O(n²), two pointers O(n) → tổng O(n³).
4. **Dedup**: Sau khi sort, `if (i > lo && nums[i] === nums[i-1]) continue` ở mỗi level.
5. **Overflow**: nums[i] có thể là ±10^9 → tổng 4 số có thể vượt int32; trong JS number OK.
6. **Follow-up**: "kSum tổng quát dùng đệ quy — giảm về 2Sum sorted ở base case."

## Solutions

```typescript
/**
 * Solution 1: Brute Force — 4 nested loops with dedup via Set
 * Time: O(n⁴) — four loops over all combinations
 * Space: O(n) — Set for dedup + result storage
 */
function fourSum1(nums: number[], target: number): number[][] {
  const n = nums.length;
  nums.sort((a, b) => a - b);
  const seen = new Set<string>();
  const result: number[][] = [];
  for (let i = 0; i < n - 3; i++)
    for (let j = i + 1; j < n - 2; j++)
      for (let k = j + 1; k < n - 1; k++)
        for (let l = k + 1; l < n; l++)
          if (nums[i] + nums[j] + nums[k] + nums[l] === target) {
            const key = `${nums[i]},${nums[j]},${nums[k]},${nums[l]}`;
            if (!seen.has(key)) {
              seen.add(key);
              result.push([nums[i], nums[j], nums[k], nums[l]]);
            }
          }
  return result;
}

/**
 * Solution 2: Sort + Fix 2 Pointers + Two Pointers (optimal)
 * Time: O(n³) — O(n log n) sort + O(n²) outer × O(n) two-pointer inner
 * Space: O(1) extra (excluding output array)
 *
 * Key: after sorting, skip duplicates at each level to avoid repeated quadruplets
 */
function fourSum(nums: number[], target: number): number[][] {
  nums.sort((a, b) => a - b);
  const result: number[][] = [];
  const n = nums.length;

  for (let i = 0; i < n - 3; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue; // skip duplicate i
    for (let j = i + 1; j < n - 2; j++) {
      if (j > i + 1 && nums[j] === nums[j - 1]) continue; // skip duplicate j
      let lo = j + 1,
        hi = n - 1;
      while (lo < hi) {
        const sum = nums[i] + nums[j] + nums[lo] + nums[hi];
        if (sum === target) {
          result.push([nums[i], nums[j], nums[lo], nums[hi]]);
          while (lo < hi && nums[lo] === nums[lo + 1]) lo++; // skip dup lo
          while (lo < hi && nums[hi] === nums[hi - 1]) hi--; // skip dup hi
          lo++;
          hi--;
        } else if (sum < target) {
          lo++;
        } else {
          hi--;
        }
      }
    }
  }
  return result;
}

// === Test Cases ===
console.log(JSON.stringify(fourSum([1, 0, -1, 0, -2, 2], 0)));
// [[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]
console.log(JSON.stringify(fourSum([2, 2, 2, 2, 2], 8)));
// [[2,2,2,2]]
console.log(JSON.stringify(fourSum([0, 0, 0, 0], 0)));
// [[0,0,0,0]]
console.log(JSON.stringify(fourSum1([1, 0, -1, 0, -2, 2], 0)));
// same as fourSum
```

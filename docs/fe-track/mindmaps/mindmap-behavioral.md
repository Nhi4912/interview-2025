## 📋 STAR Method Cheatsheet

```
┌─────────────────────────────────────────────────────────────────┐
│                    STAR METHOD                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│   S - SITUATION (15-20%, 30 seconds)                            │
│   ─────────────────────────────────                              │
│   • Company/team context                                        │
│   • Project or initiative                                       │
│   • Stakes/why it mattered                                      │
│   • Keep it brief!                                              │
│                                                                   │
│   T - TASK (15-20%, 20 seconds)                                 │
│   ────────────────────────────                                   │
│   • YOUR specific responsibility                                │
│   • Use "I" not "we"                                            │
│   • What was expected of YOU                                    │
│                                                                   │
│   A - ACTION (50-60%, 90 seconds) ⭐ MOST IMPORTANT            │
│   ─────────────────────────────────────────────                  │
│   • What YOU specifically did                                   │
│   • WHY you took each action                                    │
│   • Multiple actions (3-5)                                      │
│   • Power verbs: Led, Designed, Implemented, Convinced          │
│                                                                   │
│   R - RESULT (15-20%, 30 seconds)                               │
│   ───────────────────────────────                                │
│   • Quantified outcomes (%, $, time)                            │
│   • Business/team impact                                        │
│   • Learnings (especially for failures)                         │
│                                                                   │
│   TOTAL TIME: 2-3 minutes                                       │
│                                                                   │
└─────────────────────────────────────────────────────────────────┘
```

## 🎭 Storytelling Tips

```
┌─────────────────────────────────────────────────────────────────┐
│                    STORYTELLING ELEMENTS                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│   OPENING HOOKS:                                                │
│   ──────────────                                                 │
│   Stakes: "We had 48 hours to fix a bug affecting 100K users"   │
│   Contrast: "Everyone said it couldn't be done"                 │
│   Outcome: "This project taught me the most about leadership"   │
│   Problem: "Our checkout had 40% abandonment - nobody knew why" │
│                                                                   │
│   EMOTIONAL BEATS:                                              │
│   ────────────────                                               │
│   Tension: "I knew if we missed this deadline..."               │
│   Vulnerability: "Honestly, I was nervous about..."             │
│   Triumph: "The metrics showed 40% improvement"                 │
│   Growth: "That experience taught me..."                        │
│                                                                   │
│   SPECIFICITY:                                                  │
│   ────────────                                                   │
│   ❌ "I improved performance"                                   │
│   ✅ "I reduced load time from 8s to 2.4s, decreasing           │
│       bounce rate by 35%"                                       │
│                                                                   │
│   COMMON MISTAKES:                                              │
│   ────────────────                                               │
│   • Too much "we" not enough "I"                                │
│   • Skipping to the end                                         │
│   • No concrete results                                         │
│   • Blaming others                                              │
│   • Going too long (>4 min)                                     │
│                                                                   │
└─────────────────────────────────────────────────────────────────┘
```

## ✅ Interview Checklist

```
BEFORE INTERVIEW:
□ Research company's specific LPs/values
□ Prepare 5 essential stories
□ Practice each story (2-3 min)
□ Practice adapting stories to different angles
□ Prepare questions for interviewer
□ Test video/audio setup

DURING INTERVIEW:
□ Listen carefully to the exact question
□ Take a breath before answering
□ Use STAR structure
□ Focus on "I" not "we"
□ Include specific numbers/details
□ Watch your time (2-3 min)
□ End with learning/reflection

AFTER EACH ANSWER:
□ Check if they have follow-up questions
□ Offer to elaborate if they seem interested

COMMON FOLLOW-UPS:
• "What would you do differently?"
• "What did you learn?"
• "How did others react?"
• "What was most challenging?"
```

## 📝 Quick Answer Frameworks

```
WEAKNESS:
"I tend to [real weakness]. I've addressed this by [action].
For example, [specific example of improvement]."

WHY THIS COMPANY:
"I'm excited about [company] because [specific about product].
This role fits my experience in [skill] and interest in [growth area].
I want to contribute to [specific initiative]."

FAILURE:
Own it → What happened → What you learned → How you apply it now

CONFLICT:
Both sides fairly → How you listened → Common ground → Resolution
→ Relationship now
```

## 🗓️ Preparation Timeline

```
1 WEEK BEFORE:
□ Finalize 5 stories
□ Map stories to company LPs
□ Practice with timer

3 DAYS BEFORE:
□ Mock interview with partner
□ Refine based on feedback
□ Practice hooks and endings

1 DAY BEFORE:
□ Light review of stories
□ Review company values
□ Prepare questions to ask
□ Get good sleep

INTERVIEW DAY:
□ Arrive/log in early
□ Have notes nearby (not to read, to glance)
□ Water nearby
□ Deep breaths before starting
```

## 📚 Key Formulas

```
STAR TIME SPLIT:
S: 30s (15-20%)
T: 20s (15-20%)
A: 90s (50-60%) ⭐
R: 30s (15-20%)

STORY BANK:
5 core stories × 3 angles each = 15 variations

ANSWER LENGTH:
Too short: <1.5 min (lacks detail)
Just right: 2-3 min ✅
Too long: >4 min (loses interviewer)

PRACTICE RATIO:
For every 1 hour of interview, practice 3-5 hours
```

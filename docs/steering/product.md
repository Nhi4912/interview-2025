# Product Steering Document

## Problem Statement
**Problem:** Developers preparing for technical interviews at top tech companies lack a structured, bilingual (EN/VI) study resource that covers both Frontend and Backend tracks with a clear progression path.
**Who faces this:** Vietnamese developers targeting Mid/Senior roles at Zalo (VNG), Grab, Axon, Employment Hero, Microsoft, Google.
**Current solutions:** Scattered resources (LeetCode, System Design Primer, random blog posts) — no unified platform, no Vietnamese explanations, no track separation.

## Product Vision
> Interview 2025 will be the go-to bilingual interview preparation platform for Vietnamese developers targeting Frontend and Backend roles at top tech companies.

## Target Users
### Primary Persona: Minh — Mid-level Developer
- Age: 25-30
- Role: 2-4 years experience, Frontend or Backend
- Pain points: Doesn't know what to study, overwhelmed by English-only resources
- Goals: Land a job at Grab/Zalo/Microsoft within 3-6 months
- Tech comfort: High — daily coding, but gaps in CS theory

### Secondary Persona: Hoa — Senior Developer
- Age: 28-35
- Role: 4-7 years experience, wants to level up
- Pain points: Strong in daily work but weak in system design interviews
- Goals: Pass system design rounds at Google/Microsoft
- Tech comfort: Very high — needs depth, not basics

## Core Value Proposition
1. **Bilingual** — English terms + Vietnamese explanations for intuitive understanding
2. **Two tracks** — Frontend (JS/TS/React) and Backend (Go) with shared CS fundamentals
3. **Company-targeted** — Interview patterns specific to Zalo, Grab, Axon, EH, Microsoft, Google
4. **Progressive difficulty** — Junior → Middle → Senior tags on every question

## Success Metrics (v1.0)
- Content coverage: 100% of common interview topics for both tracks
- Study completion: Users can follow roadmap from start to finish
- Quality: Each topic has Q&A format with difficulty tags

## Out of Scope (v1.0)
- Online coding exercises / judge system
- User accounts / progress tracking backend
- Mobile app
- Video content

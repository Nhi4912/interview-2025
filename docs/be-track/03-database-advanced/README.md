## Study Order / Thứ Tự Học

1. Start with SQL fundamentals — ACID and transaction isolation are frequently tested / Bắt đầu với nền tảng SQL — ACID và transaction isolation thường được hỏi
2. Then indexing & query optimisation to answer performance questions confidently / Tiếp theo là indexing & tối ưu query để tự tin trả lời câu hỏi về hiệu suất
3. NoSQL, Redis, and MongoDB for polyglot persistence discussions / NoSQL, Redis và MongoDB cho các cuộc thảo luận về lưu trữ đa mô hình
4. Finish with caching patterns — ties SQL and NoSQL knowledge together / Kết thúc với caching pattern — kết nối kiến thức SQL và NoSQL

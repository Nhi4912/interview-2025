## Study Order / Thứ Tự Học

1. Start with language fundamentals / Bắt đầu với cú pháp và kiểu dữ liệu cơ bản
2. Then interfaces & generics to understand Go's type system / Tiếp theo là interface & generics để hiểu hệ thống kiểu
3. Concurrency is Go's biggest differentiator — study it thoroughly / Concurrency là điểm mạnh của Go — học kỹ phần này
4. Memory & GC for performance-focused interviews / Bộ nhớ & GC cho các buổi phỏng vấn về hiệu suất
5. Testing & profiling for production-readiness discussions / Testing & profiling cho các câu hỏi về production
6. Data structures & algorithms for coding rounds / Cấu trúc dữ liệu & thuật toán cho vòng coding
7. Finish with advanced patterns for senior-level questions / Kết thúc với các pattern nâng cao cho câu hỏi senior

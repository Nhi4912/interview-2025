## Study Order / Thứ Tự Học

1. Start with the design framework to build a repeatable interview structure / Bắt đầu với framework thiết kế để xây dựng cấu trúc phỏng vấn lặp lại được
2. Work through classic problems to internalise common design patterns / Thực hành các bài toán kinh điển để nắm vững các design pattern phổ biến
3. Distributed patterns to handle scale and fault-tolerance questions / Các pattern phân tán để xử lý câu hỏi về mở rộng và chịu lỗi
4. Observability & scale for senior-level and staff-level discussions / Observability & scale cho thảo luận cấp senior và staff
5. Advanced problems to stretch into ambiguous, open-ended scenarios / Các bài toán nâng cao để thực hành các tình huống mơ hồ, mở
6. Finish with the ride-hailing case study as a full end-to-end practice / Kết thúc với case study gọi xe như một bài thực hành toàn diện

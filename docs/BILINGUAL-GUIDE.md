# Bilingual Guide / Hướng Dẫn Song Ngữ
## English & Vietnamese / Tiếng Anh & Tiếng Việt

---

## About This Guide / Về Hướng Dẫn Này

**English:** This frontend interview preparation guide is designed to be fully bilingual, providing content in both English and Vietnamese. This approach helps Vietnamese developers prepare for international tech interviews while learning in their native language.

**Tiếng Việt:** Hướng dẫn chuẩn bị phỏng vấn frontend này được thiết kế hoàn toàn song ngữ, cung cấp nội dung bằng cả tiếng Anh và tiếng Việt. Cách tiếp cận này giúp các lập trình viên Việt Nam chuẩn bị cho phỏng vấn công nghệ quốc tế trong khi học bằng ngôn ngữ mẹ đẻ của họ.

---

## Structure / Cấu Trúc

### Headings / Tiêu Đề

All major headings include both languages:

Tất cả tiêu đề chính bao gồm cả hai ngôn ngữ:

```markdown
# Main Title / Tiêu Đề Chính
## Subtitle / Phụ Đề
### Section / Phần
```

### Definitions / Định Nghĩa

Key concepts are explained in both languages:

Các khái niệm chính được giải thích bằng cả hai ngôn ngữ:

```markdown
**English:** Definition in English...

**Tiếng Việt:** Định nghĩa bằng tiếng Việt...
```

### Code Comments / Chú Thích Code

```javascript
// English comment / Chú thích tiếng Việt
const example = 'value';

// Important note / Ghi chú quan trọng
function doSomething() {
  // Implementation / Triển khai
}
```

### Interview Questions / Câu Hỏi Phỏng Vấn

```markdown
### Q1: Question in English?
### C1: Câu hỏi bằng tiếng Việt?

**English Answer:** Answer in English...

**Câu Trả Lời Tiếng Việt:** Câu trả lời bằng tiếng Việt...
```

## How to Use / Cách Sử Dụng

### Reading Strategy / Chiến Lược Đọc

**English:**
1. Read English version first for technical accuracy
2. Check Vietnamese version for clarity
3. Compare both to reinforce understanding
4. Practice explaining concepts in both languages

**Tiếng Việt:**
1. Đọc phiên bản tiếng Anh trước để chính xác kỹ thuật
2. Kiểm tra phiên bản tiếng Việt để rõ ràng
3. So sánh cả hai để củng cố hiểu biết
4. Thực hành giải thích khái niệm bằng cả hai ngôn ngữ

### Study Tips / Mẹo Học Tập

**English:**
- Use English for code and technical terms
- Use Vietnamese for concept understanding
- Practice interviews in English
- Take notes in both languages

**Tiếng Việt:**
- Sử dụng tiếng Anh cho code và thuật ngữ kỹ thuật
- Sử dụng tiếng Việt để hiểu khái niệm
- Thực hành phỏng vấn bằng tiếng Anh
- Ghi chú bằng cả hai ngôn ngữ

## Contributing / Đóng Góp

### Translation Guidelines / Hướng Dẫn Dịch

**English:**
1. Maintain technical accuracy
2. Use consistent terminology
3. Keep code examples in English
4. Translate explanations naturally
5. Preserve formatting and structure

**Tiếng Việt:**
1. Duy trì độ chính xác kỹ thuật
2. Sử dụng thuật ngữ nhất quán
3. Giữ ví dụ code bằng tiếng Anh
4. Dịch giải thích một cách tự nhiên
5. Bảo toàn định dạng và cấu trúc

### Quality Standards / Tiêu Chuẩn Chất Lượng

**English:**
- Technical terms should be accurate
- Grammar should be correct in both languages
- Examples should be clear and practical
- Explanations should be concise
- Code should follow best practices

**Tiếng Việt:**
- Thuật ngữ kỹ thuật phải chính xác
- Ngữ pháp phải đúng ở cả hai ngôn ngữ
- Ví dụ phải rõ ràng và thực tế
- Giải thích phải ngắn gọn
- Code phải tuân theo thực hành tốt nhất

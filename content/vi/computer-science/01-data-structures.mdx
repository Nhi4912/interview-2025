# Cấu trúc dữ liệu

## Giới thiệu

Cấu trúc dữ liệu là cách tổ chức và lưu trữ dữ liệu để có thể truy cập và sửa đổi hiệu quả. Hiểu rõ cấu trúc dữ liệu là nền tảng quan trọng cho lập trình viên và là chủ đề phổ biến trong phỏng vấn kỹ thuật.

## Tại sao cấu trúc dữ liệu quan trọng?

1. **Hiệu suất**: Chọn đúng cấu trúc dữ liệu cải thiện hiệu suất
2. **Tổ chức code**: Code dễ hiểu và maintain hơn
3. **Giải quyết vấn đề**: Nhiều thuật toán phụ thuộc vào cấu trúc dữ liệu
4. **Phỏng vấn**: Chủ đề quan trọng trong phỏng vấn kỹ thuật

## Arrays (Mảng)

Arrays là cấu trúc dữ liệu cơ bản nhất, lưu trữ các phần tử liên tiếp trong bộ nhớ.

<CodeExample title="Array Operations" language="javascript">
```javascript
// Tạo array
const numbers = [1, 2, 3, 4, 5];

// Truy cập phần tử - O(1)
console.log(numbers[0]); // 1

// Thêm phần tử cuối - O(1)
numbers.push(6);

// Xóa phần tử cuối - O(1)
numbers.pop();

// Thêm phần tử đầu - O(n)
numbers.unshift(0);

// Xóa phần tử đầu - O(n)
numbers.shift();

// Tìm kiếm - O(n)
const index = numbers.indexOf(3);

// Duyệt array - O(n)
numbers.forEach(num => console.log(num));

````
</CodeExample>

### Độ phức tạp của Array

| Thao tác | Độ phức tạp |
|----------|-------------|
| Truy cập | O(1) |
| Tìm kiếm | O(n) |
| Thêm cuối | O(1) |
| Thêm đầu | O(n) |
| Xóa cuối | O(1) |
| Xóa đầu | O(n) |

## Linked Lists (Danh sách liên kết)

Linked list là một chuỗi các nodes, mỗi node chứa dữ liệu và tham chiếu đến node tiếp theo.

<CodeExample title="Linked List Implementation" language="javascript">
```javascript
class Node {
  constructor(data) {
    this.data = data;
    this.next = null;
  }
}

class LinkedList {
constructor() {
this.head = null;
this.size = 0;
}

// Thêm node vào đầu - O(1)
prepend(data) {
const newNode = new Node(data);
newNode.next = this.head;
this.head = newNode;
this.size++;
}

// Thêm node vào cuối - O(n)
append(data) {
const newNode = new Node(data);

    if (!this.head) {
      this.head = newNode;
    } else {
      let current = this.head;
      while (current.next) {
        current = current.next;
      }
      current.next = newNode;
    }
    this.size++;

}

// Xóa node - O(n)
remove(data) {
if (!this.head) return;

    if (this.head.data === data) {
      this.head = this.head.next;
      this.size--;
      return;
    }

    let current = this.head;
    while (current.next) {
      if (current.next.data === data) {
        current.next = current.next.next;
        this.size--;
        return;
      }
      current = current.next;
    }

}

// Tìm kiếm - O(n)
find(data) {
let current = this.head;
while (current) {
if (current.data === data) {
return current;
}
current = current.next;
}
return null;
}

// In list
print() {
let current = this.head;
const values = [];
while (current) {
values.push(current.data);
current = current.next;
}
console.log(values.join(' -> '));
}
}

// Sử dụng
const list = new LinkedList();
list.append(1);
list.append(2);
list.append(3);
list.prepend(0);
list.print(); // 0 -> 1 -> 2 -> 3

````
</CodeExample>

## Stacks (Ngăn xếp)

Stack là cấu trúc dữ liệu LIFO (Last In, First Out) - phần tử cuối cùng vào sẽ ra đầu tiên.

<CodeExample title="Stack Implementation" language="javascript">
```javascript
class Stack {
  constructor() {
    this.items = [];
  }

// Thêm phần tử - O(1)
push(element) {
this.items.push(element);
}

// Xóa và trả về phần tử trên cùng - O(1)
pop() {
if (this.isEmpty()) {
return null;
}
return this.items.pop();
}

// Xem phần tử trên cùng - O(1)
peek() {
if (this.isEmpty()) {
return null;
}
return this.items[this.items.length - 1];
}

// Kiểm tra stack rỗng - O(1)
isEmpty() {
return this.items.length === 0;
}

// Lấy kích thước - O(1)
size() {
return this.items.length;
}

// Xóa tất cả - O(1)
clear() {
this.items = [];
}
}

// Ví dụ: Kiểm tra dấu ngoặc hợp lệ
function isValidParentheses(str) {
const stack = new Stack();
const pairs = {
'(': ')',
'[': ']',
'{': '}'
};

for (let char of str) {
if (pairs[char]) {
stack.push(char);
} else if (Object.values(pairs).includes(char)) {
if (stack.isEmpty() || pairs[stack.pop()] !== char) {
return false;
}
}
}

return stack.isEmpty();
}

console.log(isValidParentheses("()[]{}")); // true
console.log(isValidParentheses("([)]")); // false
console.log(isValidParentheses("{[()]}")); // true

````
</CodeExample>

<CodeExample title="Queue Implementation" language="javascript">
```javascript
class Queue {
  constructor() {
    this.items = [];
  }

// Thêm phần tử vào cuối - O(1)
enqueue(element) {
this.items.push(element);
}

// Xóa và trả về phần tử đầu - O(n)
dequeue() {
if (this.isEmpty()) {
return null;
}
return this.items.shift();
}

// Xem phần tử đầu - O(1)
front() {
if (this.isEmpty()) {
return null;
}
return this.items[0];
}

// Kiểm tra queue rỗng - O(1)
isEmpty() {
return this.items.length === 0;
}

// Lấy kích thước - O(1)
size() {
return this.items.length;
}
}

// Ví dụ: BFS (Breadth-First Search)
function bfs(graph, start) {
const queue = new Queue();
const visited = new Set();
const result = [];

queue.enqueue(start);
visited.add(start);

while (!queue.isEmpty()) {
const node = queue.dequeue();
result.push(node);

    for (let neighbor of graph[node]) {
      if (!visited.has(neighbor)) {
        visited.add(neighbor);
        queue.enqueue(neighbor);
      }
    }

}

return result;
}

````
</CodeExample>

## Hash Tables (Bảng băm)

Hash table lưu trữ key-value pairs và cung cấp truy cập nhanh thông qua hash function.

<CodeExample title="Hash Table Implementation" language="javascript">
```javascript
class HashTable {
  constructor(size = 53) {
    this.keyMap = new Array(size);
  }

  // Hash function
  _hash(key) {
    let total = 0;
    const PRIME = 31;

    for (let i = 0; i < Math.min(key.length, 100); i++) {
      const char = key[i];
      const value = char.charCodeAt(0) - 96;
      total = (total * PRIME + value) % this.keyMap.length;
    }

    return total;
  }

  // Set key-value - O(1) trung bình
  set(key, value) {
    const index = this._hash(key);

    if (!this.keyMap[index]) {
      this.keyMap[index] = [];
    }

    // Kiểm tra key đã tồn tại
    for (let i = 0; i < this.keyMap[index].length; i++) {
      if (this.keyMap[index][i][0] === key) {
        this.keyMap[index][i][1] = value;
        return;
      }
    }

    this.keyMap[index].push([key, value]);
  }

  // Get value - O(1) trung bình
  get(key) {
    const index = this._hash(key);

    if (this.keyMap[index]) {
      for (let i = 0; i < this.keyMap[index].length; i++) {
        if (this.keyMap[index][i][0] === key) {
          return this.keyMap[index][i][1];
        }
      }
    }

    return undefined;
  }

  // Lấy tất cả keys
  keys() {
    const keysArr = [];

    for (let i = 0; i < this.keyMap.length; i++) {
      if (this.keyMap[i]) {
        for (let j = 0; j < this.keyMap[i].length; j++) {
          keysArr.push(this.keyMap[i][j][0]);
        }
      }
    }

    return keysArr;
  }

  // Lấy tất cả values
  values() {
    const valuesArr = [];

    for (let i = 0; i < this.keyMap.length; i++) {
      if (this.keyMap[i]) {
        for (let j = 0; j < this.keyMap[i].length; j++) {
          if (!valuesArr.includes(this.keyMap[i][j][1])) {
            valuesArr.push(this.keyMap[i][j][1]);
          }
        }
      }
    }

    return valuesArr;
  }
}

// Sử dụng
const ht = new HashTable();
ht.set("name", "Nguyễn Văn A");
ht.set("age", 25);
ht.set("city", "Hà Nội");

console.log(ht.get("name")); // Nguyễn Văn A
console.log(ht.keys());      // ["name", "age", "city"]
````

</CodeExample>

## Trees (Cây)

Tree là cấu trúc dữ liệu phân cấp với root node và các child nodes.

</Diagram>

<CodeExample title="Binary Search Tree" language="javascript">
```javascript
class TreeNode {
  constructor(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}

class BinarySearchTree {
constructor() {
this.root = null;
}

// Thêm node - O(log n) trung bình
insert(value) {
const newNode = new TreeNode(value);

    if (!this.root) {
      this.root = newNode;
      return this;
    }

    let current = this.root;
    while (true) {
      if (value === current.value) return undefined;

      if (value < current.value) {
        if (!current.left) {
          current.left = newNode;
          return this;
        }
        current = current.left;
      } else {
        if (!current.right) {
          current.right = newNode;
          return this;
        }
        current = current.right;
      }
    }

}

// Tìm kiếm - O(log n) trung bình
find(value) {
if (!this.root) return false;

    let current = this.root;
    while (current) {
      if (value === current.value) return true;
      if (value < current.value) {
        current = current.left;
      } else {
        current = current.right;
      }
    }

    return false;

}

// Duyệt In-order (Left -> Root -> Right)
inOrder(node = this.root, result = []) {
if (node) {
this.inOrder(node.left, result);
result.push(node.value);
this.inOrder(node.right, result);
}
return result;
}

// Duyệt Pre-order (Root -> Left -> Right)
preOrder(node = this.root, result = []) {
if (node) {
result.push(node.value);
this.preOrder(node.left, result);
this.preOrder(node.right, result);
}
return result;
}

// Duyệt Post-order (Left -> Right -> Root)
postOrder(node = this.root, result = []) {
if (node) {
this.postOrder(node.left, result);
this.postOrder(node.right, result);
result.push(node.value);
}
return result;
}
}

// Sử dụng
const bst = new BinarySearchTree();
bst.insert(10);
bst.insert(5);
bst.insert(15);
bst.insert(3);
bst.insert(7);

console.log(bst.find(7)); // true
console.log(bst.inOrder()); // [3, 5, 7, 10, 15]
console.log(bst.preOrder()); // [10, 5, 3, 7, 15]
console.log(bst.postOrder()); // [3, 7, 5, 15, 10]

```

## So sánh các cấu trúc dữ liệu

| Cấu trúc | Truy cập | Tìm kiếm | Thêm | Xóa | Ưu điểm | Nhược điểm |
|----------|----------|----------|------|-----|---------|------------|
| Array | O(1) | O(n) | O(n) | O(n) | Truy cập nhanh | Thêm/xóa chậm |
| Linked List | O(n) | O(n) | O(1) | O(1) | Thêm/xóa nhanh | Truy cập chậm |
| Stack | O(n) | O(n) | O(1) | O(1) | LIFO đơn giản | Truy cập giới hạn |
| Queue | O(n) | O(n) | O(1) | O(1) | FIFO đơn giản | Truy cập giới hạn |
| Hash Table | O(1) | O(1) | O(1) | O(1) | Rất nhanh | Cần bộ nhớ nhiều |
| BST | O(log n) | O(log n) | O(log n) | O(log n) | Tìm kiếm nhanh | Có thể mất cân bằng |

## Câu hỏi phỏng vấn thường gặp

### Câu 1: Khi nào nên dùng Array vs Linked List?

**Trả lời:**
- **Array**: Khi cần truy cập ngẫu nhiên nhanh, biết trước kích thước
- **Linked List**: Khi thường xuyên thêm/xóa, kích thước thay đổi động

### Câu 2: Sự khác biệt giữa Stack và Queue?

**Trả lời:**
- **Stack**: LIFO (Last In, First Out) - như chồng đĩa
- **Queue**: FIFO (First In, First Out) - như hàng đợi

### Câu 3: Hash Table hoạt động như thế nào?

**Trả lời:**
Hash table sử dụng hash function để chuyển key thành index trong array. Collision được xử lý bằng chaining hoặc open addressing.

### Câu 4: Binary Search Tree có gì đặc biệt?

**Trả lời:**
BST có tính chất: node bên trái < root < node bên phải. Điều này cho phép tìm kiếm O(log n) trong trường hợp cân bằng.

### Câu 5: Khi nào nên dùng Hash Table?

**Trả lời:**
Dùng Hash Table khi:
- Cần truy cập O(1)
- Không quan tâm thứ tự
- Có đủ bộ nhớ
- Keys là unique

## Tóm tắt

- Arrays: Truy cập nhanh, thêm/xóa chậm
- Linked Lists: Thêm/xóa nhanh, truy cập chậm
- Stacks: LIFO, dùng cho undo/redo, DFS
- Queues: FIFO, dùng cho BFS, task scheduling
- Hash Tables: Truy cập O(1), dùng cho caching, indexing
- Trees: Cấu trúc phân cấp, BST cho tìm kiếm nhanh
